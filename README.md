# 💼 Portafolio Personal – Francisco Angarita

Este proyecto es mi portafolio profesional como desarrollador Full-Stack Junior. Aquí presento mis habilidades, herramientas y algunos de los proyectos en los que he trabajado, siguiendo buenas prácticas de desarrollo web.

## 🚀 Descripción

El sitio muestra información sobre mi perfil profesional, tecnologías que manejo y proyectos destacados. Está diseñado con un enfoque en claridad visual, estructura semántica y adaptabilidad a diferentes tamaños de pantalla.

## 🛠️ Tecnologías utilizadas

- HTML5 semántico
- CSS3 (Flexbox y diseño responsive)
- Metodología BEM para organización de clases
- Git y GitHub para control de versiones
- Figma como herramienta de diseño

## 📂 Estructura del proyecto

- `index.html` → estructura principal del sitio
- `styles/` → archivos CSS
- `images/` → imágenes e íconos
- `files/` → documentos como el CV
